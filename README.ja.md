# ChatLogVault

AIチャットログを構造化された再利用可能なナレッジに変換するツール。

ChatLogVaultは、ChatGPTなどのAIチャットログをナレッジとして管理するための個人向けツールです。  
チャット内容を保存・整理・再構成し、再利用可能な知識として活用できます。

---

## 特徴

- API経由でチャットログ取り込み（Tampermonkey → API）
- フォルダ（階層構造）・タグによる整理
- お気に入り管理
- メッセージの表示/非表示切り替え
- メッセージのピン留め
- 見出し付与による目次生成
- 表示モード切替（チャット風 / Markdown / JSON）
- 画像の分離保存（UUID管理）
- i18n対応（ブラウザ言語 + 英語フォールバック）

---

## インポート方法

ChatLogVaultは以下の方法でチャットログを取り込むことができます。

- AIチャットサービスからエクスポートしたJSONファイルのアップロード
- API経由での送信
- Tampermonkeyスクリプトを利用してChatGPTの会話を直接送信

### Tampermonkey連携

Tampermonkeyのユーザースクリプトを使用することで、
ChatGPTのチャットログを抽出し、API経由でChatLogVaultへ直接送信できます。

手動でファイルを扱うことなく、ワンクリックでインポートが可能です。

---

## コンセプト

ChatLogVaultは単なるログビューアではありません。

👉 **チャットを整理・再構成してナレッジ化するツール**

- JSONを正本として扱う
- UI状態（非表示・ピン・見出し）は別管理
- チャットを「読みやすい形」に再構成できる

## アーキテクチャ

- 一意キー：`source + url`
- インポート時は上書き（最新状態を保持）
- メッセージ状態は別テーブルで管理
- 画像はBase64から抽出して外部保存
- Markdown内はプレースホルダーで参照
